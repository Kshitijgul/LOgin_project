# Login and Register WEBSITE BY MERN STACK 

## Features


- Uses MERN Stack For the complete website 
- For Styling Uses Tailwind CSS 

## Installation

Login_project requires [Node.js](https://nodejs.org/) v10+ to run.


## _Frontend_
```sh

cd Login_project
cd frontend
npm i
node index.js
```


## _Backend_
```sh
cd Login_project
cd backend
npm i
node index.js
```
